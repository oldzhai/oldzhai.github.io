二.程序

我测试的局域网处于零网段的所以ip地址为 192.168.0.103，如果是两根网线直连我们只要保持在同一网段就可以通讯，如果在一台主机上测试用 127.0.0.1 是内网ip

1.服务器

tcsocket—>bind—>listen—–>accept—>io函数（send/recv read/write）

  1. 创建一个网络的软通道—–》socket 函数 （造作系统提供的）（流式套接字）
    创建成功：返回文件描述符
    socket 函数
    参数 一 ：协议域 （ipv4）AF_INET 网络通道
    二 ： 类型 （流逝套接字） SOCK_STREAM
    三 ：0 （自动匹配相关协议）
    返回值：
    成功：返回文件描述符 （fd）—–（询问的通道）
    失败： -1；

  2. 绑定一个网络的终端 —》bind （配置自己的网络）
    bind函数
    参数一：文件描述符
    二：绑定的地址
    三： 结构体的长度
    返回值： 成功 0

  3. 网络监听 listen
    listen 函数
    参数一： 文件描述符
    二： 真听的最大个数 （默认 5）
    返回值： 成功 0
    失败 -1

  4. 接收客户端连接请求 accept （接收后保存客户端的网络地址）
    accept函数
    参数一 ：文件描述符（套接字）
    二 ： 用于保存客户端的地址
    三 ：地址的长度
    返回值：
    成功： 非负整数 新的文件描述符 （进行正文数据的收发）
    失败： -1

  5. 收发（io函数收发） （send/recv）（read/write）
    recv 函数
    参数一： 新的文件描述符
    二： 定义的数组的 buff
    三：buf的大小 （最大能收多少）
    四： flags 0 （阻塞接收）
    返回值： 成功：实际接收的字节数
    失败 -1
    send 函数 （发送）
    参数一： 新的文件描述符
    二： 定义的数组的 buff
    三：buf的大小 （最大能收多少）
    四： flags 0 （阻塞接收）
    返回值： 成功：实际发送的字节数
    失败 -1

2.客户端

socket—>connect—>io函数（send/recv read/write）

  1. socket 创建套接字（流式套接字）
  2. connect 建立连接
    connect函数
    参数一： 文件描述符
    二： 服务器的地址
    三： 地址结构体的长度
    返回值： 成功 0
    失败 -1
  3. io 函数 （收发函数）
    send 发送 recv接收
