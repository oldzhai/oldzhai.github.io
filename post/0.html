vsCode配置

一.简介

最近发现了一个挺不错的软件vsCode，之前使用 visual studio 由于占的内存空间比较大，每次启动都不是很方便，对于vsCode 这个比较小巧的软件，内部集成了各种插件，根据编程的不同来选择配置，在这里推荐给大家

二.软件下载与语言配置

1.注意，安装很简单，傻瓜式的安装就可以了

下载地址：vsCode下载

2.我们安装完成后界面是英文版的我们需要安装中文的插件

在vsCode商店搜索 chinese 直接安装，重启软件就可以显示中文了

三.vsCode配置c/c++

1.准备

先点击此处下载编译的包：Mingw-w64

2.配置环境变量

  1. 首先我们将下载好的压缩包解压到我们vsCode软件安装的目录下

  2. 打开电脑，找到环境变量的设置（可以在搜索框打上环境 进行搜索）

    注意添加完环境变量，保存退出

3.安装vsCode相关插件

  1. 我们打开vsCode ，安装下面4个插件

    • C/C++
    • C/C++ Compile Run
    • C/C++ GNU Global
    • Code Runner

4.新建测试文件

打开一个保存代码的目录，新建一个 .c 文件,写上测试代码

5.运行与调试

1.点击上面的小三角形，运行我们会发现出现了乱码

2.按F5 运行，出来次选项框，选择第一个

再次选择第一个

注意：在出现的代码界面修改下面部分（是让运行结果通过电脑的命令窗口输出），我们可以不用修改（直接在软件终端输出）

返回测试代码，按F5 运行

点击配置任务后继续点第一个

然后再运行的时候我们就可以看到运行结果了

6.单步运行调试

我们用鼠标点击代码行号前面，来设置断点，然后按F5调试，一步一步运行

7.控制台乱码

打开电脑的控制面板，找到时钟和区域，如下设置即可，设置完后需要重启电脑

四.总结
